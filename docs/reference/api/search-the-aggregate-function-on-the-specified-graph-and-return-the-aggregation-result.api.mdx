<h1
  className={"openapi__heading"}
>
  Search the aggregate function on the specified graph and return the aggregation result.
</h1>

<MethodEndpoint
  method={"post"}
  path={"/graph/{graph_id}/search/aggregate"}
>
  
</MethodEndpoint>

Search and aggregate the graph and return the resulting aggregated data. A section can be defined (defaults to `/` == root) to interpret relative property paths. Example: is(volume) and (reported.age&gt;23d or desired.clean==true or metadata.version==2)
