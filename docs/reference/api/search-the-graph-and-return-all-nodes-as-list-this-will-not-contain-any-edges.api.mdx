<h1
  className={"openapi__heading"}
>
  Search the graph and return all nodes as list (this will not contain any edges)
</h1>

<MethodEndpoint
  method={"post"}
  path={"/graph/{graph_id}/search/list"}
>
  
</MethodEndpoint>

Search the graph and return the matching nodes as list. A section can be defined (defaults to `/` == root) to interpret relative property paths. Example: is(volume) and (reported.age&gt;23d or desired.clean==true or metadata.version==2)
