<h1
  className={"openapi__heading"}
>
  Search the graph and return the resulting graph.
</h1>

<MethodEndpoint
  method={"post"}
  path={"/graph/{graph_id}/search/graph"}
>
  
</MethodEndpoint>

Search the graph and return the matching nodes including the traversed edges. The resulting data can be interpreted as a graph. A section can be defined (defaults to `/` == root) to interpret relative property paths. Example: is(volume) and (reported.age&gt;23d or desired.clean==true or metadata.version==2)
