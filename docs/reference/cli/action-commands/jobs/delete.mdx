# `jobs delete` Command

The `jobs delete` command deletes a job.

## Usage

```bash
jobs delete <id>
```

### Parameters

| Parameter | Description    | Required? | Default Value |
| --------- | -------------- | --------- | ------------- |
| `id`      | Job identifier | ✔️        | N/A           |

## Examples

```bash
> jobs delete say-hello
# highlight-next-line
​Job say-hello deleted.
```
