# `jobs` Command

The `jobs` command allows for the management of jobs.

| Command                               | Description                                                                                |
| ------------------------------------- | ------------------------------------------------------------------------------------------ |
| [`jobs activate`](./activate.mdx)     | Activate the triggers of a job                                                             |
| [`jobs add`](./add.mdx)               | Add a job to the task handler                                                              |
| [`jobs deactivate`](./deactivate.mdx) | Deactivate the triggers of a job, so that the job is not started if the triggers are fired |
| [`jobs delete`](./delete.mdx)         | Delete a job                                                                               |
| [`jobs list`](./list.mdx)             | List all jobs                                                                              |
| [`jobs run`](./run.mdx)               | Run a job                                                                                  |
| [`jobs running`](./running.mdx)       | List all currently running jobs                                                            |
| [`jobs show`](./show.mdx)             | Show the definition of a job                                                               |
| [`jobs update`](./update.mdx)         | Update a job                                                                               |
