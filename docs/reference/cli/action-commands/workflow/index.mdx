# `workflow` Command

The `workflow` command allows for the management of workflows.

| Command                             | Description                              |
| ----------------------------------- | ---------------------------------------- |
| [`workflow history`](./history.mdx) | Show the aggregated history of workflows |
| [`workflow list`](./list.mdx)       | List all workflows                       |
| [`workflow log`](./log.mdx)         | Show the logs of the workflow run        |
| [`workflow run`](./run.mdx)         | Run a workflow                           |
| [`workflow running`](./running.mdx) | List all currently running workflows     |
| [`workflow show`](./show.mdx)       | Show the definition of a workflow        |

### Aliases

- `workflows`
