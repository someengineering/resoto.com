# `workflows` Command

The `workflows` command allows for the management of workflows.

| Command                              | Description                              |
| ------------------------------------ | ---------------------------------------- |
| [`workflows history`](./history.mdx) | Show the aggregated history of workflows |
| [`workflows list`](./list.mdx)       | List all workflows                       |
| [`workflows log`](./log.mdx)         | Show the logs of the workflow run        |
| [`workflows run`](./run.mdx)         | Run a workflow                           |
| [`workflows running`](./running.mdx) | List all currently running workflows     |
| [`workflows show`](./show.mdx)       | Show the definition of a workflow        |

### Aliases

- `workflow`
