# `graph list` Command

The `graph list` command lists all [Resoto Rewind](/rewind) [asset inventory graph](../../../concepts/asset-inventory-graph/index.mdx) snapshots.

## Example

```bash
> graph list
# highlight-start
resoto
snapshot-resoto-hourly-20230614T222121Z
snapshot-resoto-daily-20230614T222121Z
# highlight-end
```
