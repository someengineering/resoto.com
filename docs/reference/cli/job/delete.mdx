# `job delete` Command

The `job delete` command deletes a job.

## Usage

```bash
job delete <id>
```

### Parameters

| Parameter | Description    | Required? | Default Value |
| --------- | -------------- | --------- | ------------- |
| `id`      | Job identifier | ✔️        |               |

## Examples

```bash
> jobs delete say-hello
# highlight-next-line
​Job say-hello deleted.
```
