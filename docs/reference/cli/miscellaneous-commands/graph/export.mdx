# `graph export` Command

The `graph export` command exports an infrastructure graph to disk.

## Usage

```bash
graph export [--force] [graph_name] <file_name>
```

## Example

```bash
> graph export resoto resoto.graph
# highlight-start
Received a file resoto.graph, which is stored to ./resoto.graph
# highlight-end
```
