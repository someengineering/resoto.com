# `graph export` Command

The `graph export` command exports a [Resoto Rewind](/rewind) [asset inventory graph](../../../../concepts/asset-inventory-graph/index.mdx) snapshot to disk.

## Usage

```bash
graph export [--force] [graph_name] <file_name>
```

## Example

```bash
> graph export resoto resoto.graph
# highlight-start
Received a file resoto.graph, which is stored to ./resoto.graph
# highlight-end
```
