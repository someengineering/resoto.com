# `graph list` Command

The `graph list` command lists all infrastructure graphs.

## Usage

```bash
graph list
```

## Example

```bash
> graph list
# highlight-start
resoto
snapshot-resoto-hourly-20230614T222121Z
snapshot-resoto-daily-20230614T222121Z
# highlight-end
```
