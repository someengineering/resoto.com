# `report benchmark run` Command

The `report benchmark run` command performs the named benchmark on the defined cloud accounts.

## Usage

```bash
report benchmark run <benchmark-id> [--accounts <account-id>] [--severity <level>] [--only-failing]
```

### Parameters

| Parameter    | Description                                                                                                                           | Required? | Default Value |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------- | --------- | ------------- |
| `--accounts` | List of account ids (space separated) to run the benchmark against.<br/> If not specified, the benchmark is run against all accounts. | ️         |               |
| `--severity` | Severity level to filter checks. One of info, low, medium, high, critical                                                             | ️         |               |

### Options

| Option                 | Description                                                                      | Required? | Default Value |
| ---------------------- | -------------------------------------------------------------------------------- | --------- | ------------- |
| `--only-failing`       | Only show failing checks                                                         | ️         |               |
| `--only-check-results` | Only include the result of the checks in the report, not the benchmark sections. | ️         |               |

## Example

```bash
> report benchmark run aws_cis_1_5 --severity critical
```
