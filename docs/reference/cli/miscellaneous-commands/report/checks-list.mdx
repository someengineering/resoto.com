# `report checks list` Command

The `report checks list` command lists all configured checks.

## Usage

```bash
report checks list
```

## Example

```bash
> report checks list
# highlight-start
.
.
aws_cloudtrail_log_metric_filters_route_table_changes
aws_config_enabled_in_all_regions
aws_ec2_public_ip_address
.
.
# highlight-end
```
