# `report` Command

The `report` command can generate reports.

| Command                                           | Description                        |
| ------------------------------------------------- | ---------------------------------- |
| [`report benchmarks list`](./benchmarks-list.mdx) | List available benchmarks          |
| [`report benchmark show`](./benchmark-show.mdx)   | Show information about a benchmark |
| [`report benchmark run`](./benchmark-run.mdx)     | Run a benchmark                    |
| [`report checks list`](./checks-list.mdx)         | List available checks              |
| [`report check show`](./check-show.mdx)           | Show information about a check     |
| [`report check run`](./check-run.mdx)             | Run a check                        |
