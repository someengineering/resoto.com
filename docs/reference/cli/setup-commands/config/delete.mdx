# `config delete` Command

The `config delete` command deletes the specified configuration.

## Usage

```bash
config delete <id>
```

### Parameters

| Parameter | Description              | Required? | Default Value |
| --------- | ------------------------ | --------- | ------------- |
| `id`      | Configuration identifier | ✔️        |               |

## Examples

```bash
> configs delete config_test
# highlight-next-line
​Config config_test has been deleted.
```
