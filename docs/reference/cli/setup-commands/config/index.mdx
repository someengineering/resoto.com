# `config` Command

The `config` command allows for the management of component configurations.

| Command                         | Description                                           |
| ------------------------------- | ----------------------------------------------------- |
| [`config delete`](./delete.mdx) | Delete a configuration                                |
| [`config edit`](./edit.mdx)     | Edit a configuration                                  |
| [`config list`](./list.mdx)     | List all configuration IDs                            |
| [`config set`](./set.mdx)       | Set configuration property values                     |
| [`config show`](./show.mdx)     | Show a configuration                                  |
| [`config update`](./update.mdx) | Update or create a configuration as defined in a file |

### Aliases

- `configs`
