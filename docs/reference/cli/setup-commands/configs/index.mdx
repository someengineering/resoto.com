# `configs` Command

The `configs` command allows for the management of component configurations.

| Command                          | Description                                           |
| -------------------------------- | ----------------------------------------------------- |
| [`configs delete`](./delete.mdx) | Delete a configuration                                |
| [`configs edit`](./edit.mdx)     | Edit a configuration                                  |
| [`configs list`](./list.mdx)     | List all configuration IDs                            |
| [`configs set`](./set.mdx)       | Set configuration property values                     |
| [`configs show`](./show.mdx)     | Show a configuration                                  |
| [`configs update`](./update.mdx) | Update or create a configuration as defined in a file |

### Aliases

- `config`
