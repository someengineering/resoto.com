# `user role add` Command

The `user role add` command adds a role to an existing user.

:::note

Assigned roles currently do not have any effect on user permissions.

:::

## Usage

```bash
> user role add <email> <role>
```

### Parameters

| Parameter | Description        | Required? | Default Value |
| --------- | ------------------ | --------- | ------------- |
| `email`   | User email address | ✔️        |               |
| `role`    | User role          | ✔️        |               |

## Examples

```bash
> user role add matthias@some.engineering admin
# highlight-start
​email: matthias@some.engineering
​ fullname: Matthias Veit
​ roles:
​ - read-only
​ - admin
# highlight-end
```
