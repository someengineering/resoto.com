# `user show` Command

The `user show` command prints details for a given user.

## Usage

```bash
> user show <email>
```

## Examples

```bash
> user show matthias@some.engineering
# highlight-start
​email: matthias@some.engineering
​ fullname: Matthias Veit
​ roles:
​ - read-only
# highlight-end
```
