# `system info` Command

The `system info` command prints information about the system.

## Usage

```bash
system info
```

## Examples

```bash
> system info
# highlight-start
name: resotocore
version: 2.0.0a14
cpus: 8
mem_available: 2.85 GiB
mem_total: 16.00 GiB
inside_docker: false
started_at: '2022-02-02T11:23:19Z'
# highlight-end
```
