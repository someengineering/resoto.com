# `user role delete` Command

The `user role delete` command removes a role from an existing user.

:::note

Assigned roles currently do not have any effect on user permissions.

:::

## Usage

```bash
> user role delete <email> <role>
```

### Parameters

| Parameter | Description        | Required? | Default Value |
| --------- | ------------------ | --------- | ------------- |
| `email`   | User email address | ✔️        |               |
| `role`    | User role          | ✔️        |               |

## Examples

```bash
> user role delete matthias@some.engineering admin
# highlight-start
​email: matthias@some.engineering
​ fullname: Matthias Veit
​ roles:
​ - read-only
# highlight-end
```
