# `workflows running` Command

The `workflows running` command lists all currently running workflows.

## Usage

```bash
workflows running
```

## Examples

```bash
> workflows running
# highlight-start
workflow: collect_and_cleanup
started_at: '2022-04-12T07:22:29Z'
task-id: 4feab068-ba31-11ec-abdf-dad780437c54
# highlight-end
```
