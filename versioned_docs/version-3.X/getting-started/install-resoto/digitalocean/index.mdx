# Deploy Resoto to DigitalOcean

You can deploy Resoto from the DigitalOcean Marketplace in just a few clicks.

1. Open Resoto in the DigitalOcean Marketplace:

   [![Deploy Resoto to DigitalOcean](https://mp-assets1.sfo2.digitaloceanspaces.com/deploy-to-do/do-btn-blue.svg)](https://marketplace.digitalocean.com/apps/resoto)

2. Click **Install App**.

   ![Resoto DigitalOcean Marketplace App](./img/marketplace.png)

3. Select an existing Kubernetes cluster, or create a new one. Resoto will be installed into the selected cluster.

   ![Screenshot of cluster selection modal](./img/select_cluster.png)
