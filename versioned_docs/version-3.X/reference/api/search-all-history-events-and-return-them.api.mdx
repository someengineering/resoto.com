<h1

>
  Search all history events and return them.
</h1>

<MethodEndpoint
  method={"post"}
  path={"/graph/{graph_id}/search/history/list"}
>
  
</MethodEndpoint>

Search all history events and return them. A section can be defined (defaults to `/` == root) to interpret relative property paths. Example: is(volume) and (reported.age&gt;23d or desired.clean==true or metadata.version==2)
