# `job list` Command

The `job list` command lists all jobs.

## Usage

```bash
job list
```

## Examples

```bash
> jobs list
# highlight-start
​id: say-hello
​trigger:
​  cron_expression: '* * * * *'
​command: echo hello world
# highlight-end
```
