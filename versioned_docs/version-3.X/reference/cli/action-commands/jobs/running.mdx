# `jobs running` Command

The `jobs running` command lists all currently running jobs.

## Usage

```bash
jobs running
```

## Examples

```bash
> jobs running
# highlight-start
​job: say-hello
​started_at: '2022-01-12T09:01:34Z'
​task-id: a4bb64cc-7385-11ec-b2cb-dad780437c53
# highlight-end
```
