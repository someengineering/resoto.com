# `workflows show` Command

The `workflows show` command shows the definition of a workflow.

## Usage

```bash
workflows show <id>
```

### Parameters

| Parameter | Description          | Required? | Default Value |
| --------- | -------------------- | --------- | ------------- |
| `id`      | Workflows identifier | ✔️        | N/A           |
