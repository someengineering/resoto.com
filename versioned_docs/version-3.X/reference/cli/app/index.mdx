# `app` Command

The `app` command allows for the management of infrastructure apps.

| Command                            | Description                                  |
| ---------------------------------- | -------------------------------------------- |
| [`app search`](./search.mdx)       | Search available infrastructure apps         |
| [`app info`](./info.mdx)           | Show information about an infrastructure app |
| [`app install`](./install.mdx)     | Install an infrastructure app                |
| [`app edit`](./edit.mdx)           | Edit an infrastructure app                   |
| [`app uninstall`](./uninstall.mdx) | Uninstall an infrastructure app              |
| [`app update`](./update.mdx)       | Update infrastructure apps                   |
| [`app list`](./list.mdx)           | List installed infrastructure apps.          |
| [`app run`](./run.mdx)             | Run an infrastructure app                    |

### Aliases

- `app`
