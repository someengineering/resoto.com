# `job show` Command

The `job show` command shows the definition of a job.

## Usage

```bash
job show <id>
```

### Parameters

| Parameter | Description    | Required? | Default Value |
| --------- | -------------- | --------- | ------------- |
| `id`      | Job identifier | ✔️        |               |

## Examples

```bash
> jobs show say-hello
# highlight-start
​id: say-hello
​trigger:
​  cron_expression: '* * * * *'
​command: echo hello world
# highlight-end
```
