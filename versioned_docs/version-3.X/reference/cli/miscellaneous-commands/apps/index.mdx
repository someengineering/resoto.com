# `apps` Command

The `apps` command allows for the management of infrastructure apps.

| Command                             | Description                                  |
| ----------------------------------- | -------------------------------------------- |
| [`apps search`](./search.mdx)       | Search available infrastructure apps         |
| [`apps info`](./info.mdx)           | Show information about an infrastructure app |
| [`apps install`](./install.mdx)     | Install an infrastructure app                |
| [`apps edit`](./edit.mdx)           | Edit an infrastructure app                   |
| [`apps uninstall`](./uninstall.mdx) | Uninstall an infrastructure app              |
| [`apps update`](./update.mdx)       | Update infrastructure apps                   |
| [`apps list`](./list.mdx)           | List installed infrastructure apps.          |
| [`apps run`](./run.mdx)             | Run an infrastructure app                    |

### Aliases

- `app`
