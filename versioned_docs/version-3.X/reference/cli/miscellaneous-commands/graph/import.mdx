# `graph import` Command

The `graph import` command imports an infrastructure graph from disk.

## Usage

```bash
graph import [--force] [graph_name] <file_name>
```

## Example

```bash
> graph import resoto_test resoto.graph
# highlight-start
Graph resoto_test imported from resoto.graph.
# highlight-end
```
