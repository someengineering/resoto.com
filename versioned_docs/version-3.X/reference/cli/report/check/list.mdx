# `report check list` Command

The `report check list` command lists configured checks.

## Usage

```bash
report check list
```

## Example

```bash
> report checks list
# highlight-start
.
.
aws_cloudtrail_log_metric_filters_route_table_changes
aws_config_enabled_in_all_regions
aws_ec2_public_ip_address
.
.
# highlight-end
```
