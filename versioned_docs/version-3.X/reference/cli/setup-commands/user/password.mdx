# `user password` Command

The `user password` command updates the password of an existing user.

## Usage

```bash
> user password <email> <password>
```

### Parameters

| Parameter  | Description        | Required? | Default Value |
| ---------- | ------------------ | --------- | ------------- |
| `email`    | User email address | ✔️        |               |
| `password` | User password      | ✔️        |               |

## Examples

```bash
> user password matthias@some.engineering awTJxB4vHLQ-VVisN!tc4h9_!
# highlight-start
​Password for matthias@some.engineering updated
# highlight-end
```
