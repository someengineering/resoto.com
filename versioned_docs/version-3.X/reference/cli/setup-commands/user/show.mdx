# `user show` Command

The `user show` command prints information about a user.

## Usage

```bash
> user show <email>
```

### Parameters

| Parameter | Description        | Required? | Default Value |
| --------- | ------------------ | --------- | ------------- |
| `email`   | User email address | ✔️        |               |

## Examples

```bash
> user show matthias@some.engineering
# highlight-start
​email: matthias@some.engineering
​ fullname: Matthias Veit
​ roles:
​ - read-only
# highlight-end
```
