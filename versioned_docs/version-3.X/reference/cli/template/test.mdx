# `template test` Command

The `template test` command tests a search template.

## Usage

```bash
templates test <template_values> <template_name>
```

### Parameters

| Parameter         | Description                                                           | Required?                               | Default Value |
| ----------------- | --------------------------------------------------------------------- | --------------------------------------- | ------------- |
| `template_values` | Comma-delimited list of key-value pairs, formatted as `<key>=<value>` | if template contains value placeholders |               |
| `template_name`   | Template name                                                         | ✔️                                      |               |

## Examples

```bash
> templates test kind=volume is({{kind}})
# highlight-next-line
is(volume)
```
