# `user role` Command

The `user role` command allows for the management of user roles.

:::note

Assigned roles currently do not have any effect on user permissions.

:::

| Command                            | Description                         |
| ---------------------------------- | ----------------------------------- |
| [`user role add`](./add.mdx)       | Add a role to an existing user      |
| [`user role delete`](./delete.mdx) | Delete a role from an existing user |
